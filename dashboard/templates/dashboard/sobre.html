Projeto de Monitoramento

O projeto se trata de um monitoramento em tempo real com um microcontrolador ESP8266 que utiliza sensores para captar informações como temperatura, vibração e produção de determinada máquina ou equipamento a fim de realizar um acompanhamento e sendo assim uma manutenção preventiva e até mesmo preditiva.

O Microcontrolador também é equipado com um led e um Buzzer para alertar quando alguma das leituras estiverem fora do padrão. Para a automatização programamos o microcontrolador com a plataforma Arduino IDE que utiliza como linguagem o C++, ele realiza as leituras em tempo real e envia para seu servidor web.

Para o monitoramento usamos o Django, uma interface de fácil compreensão, ele busca no servidor web as leituras dos sensores, armazena e realiza gráficos para acompanhar as leituras.

Django

O Django é um framework web de alto nível escrito em Python que incentiva o desenvolvimento rápido e o design limpo e pragmático. Ele é utilizado no projeto para criar uma interface de fácil compreensão, buscando no servidor web as leituras dos sensores, armazenando e gerando gráficos para acompanhar as leituras em tempo real.

Plotly

Plotly é uma biblioteca de gráficos interativos em Python. No projeto, é utilizada para gerar gráficos que permitem visualizar as leituras dos sensores de forma mais intuitiva e compreensível.

Pandas

O Pandas é uma biblioteca Python que oferece estruturas de dados fáceis de usar e ferramentas de análise de dados. Ele é utilizado no projeto para manipular os dados provenientes dos sensores e facilitar a análise e geração de gráficos.
